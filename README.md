# 🎯 DigiTools Platform

## 📝 Description
DigiTools is a modern, responsive digital tools marketplace built to help users discover and manage premium software tools. It features a dynamic product listing, a functional cart system, and real-time notifications for a seamless shopping experience.

## 🛠️ Technology Used
- **Frontend:** React.js (v19), JavaScript (ES6+)
- **Styling:** Tailwind CSS, DaisyUI
- **Notifications:** React-Toastify
- **Data:** JSON Local API

## 🚀 3 Key Features
1. **Dynamic Cart System:** Users can add tools to their cart with real-time price calculations and a "duplicate prevention" logic that alerts if an item is already added.
2. **Interactive UI Toggling:** Seamlessly switch between the **Product Grid** and the **Cart/Checkout** view without page reloads.
3. **Fully Responsive Design:** Optimized for all devices (Mobile, Tablet, Desktop) with a clean, professional layout inspired by Figma.
